# BloomHer Clinic Platform

BloomHer is a digital nutrition clinic platform designed to support women's health through guided assessments, personalized nutrition planning, and progress tracking.

## Features

- Patient assessment onboarding
- Goal-based nutrition planning (pregnancy, weight management, PCOS, anemia)
- Structured clinical nutrition recommendations
- Patient dashboard for progress tracking
- Clinic-style workflow (assessment → plan → follow-up)

## Tech Stack

- React (Vite or CRA)
- Tailwind CSS
- ShadCN UI components
- Future: Firebase/Supabase backend integration

## Project Structure

- Assessment module
- Nutrition plan generator
- Patient dashboard system

## Purpose

To create a scalable digital nutrition clinic system that can be used for:
- Private practice nutrition services
- Maternal health support
- Remote consultations
- Digital health entrepreneurship

## Status

Early MVP (actively in development)
